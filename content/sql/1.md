Below are the key objectives of this project:
1. Retrieve data from tables of a database using SQL SELECT statement.
2. Use different SQL operators together with the WHERE clause to set conditions on tables of a database for database manipulation.

Below is the summary of the tasks performed in this project:
- Task 1: SELECT - FROM
- Task 2: SELECT - FROM AND WHERE
- Task 3: SQL Operators
- Task 4: Wildcard Characters
- Task 5: SQL Operators - Part 2
- Task 6: Other Comparison Operators
- Task 7: SELECT DISTINCT, ORDER BY, LIMIT, and SQL Aliases
